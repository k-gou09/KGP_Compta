Obtenez votre licence 100% Gratuite

KGP Compta est gratuit. Pour des raisons de sécurité, chaque installation nécessite une clé d'activation unique liée à votre ordinateur. Suivez ce guide simple.

1

Téléchargement et Lancement

Téléchargez l'application, extraire le fichier de l'archive et lancez-la. L'application s'ouvrira directement sur le Tableau de bord.

Pour l'instant, l'application est en mode "verrouillé".

2

Trouver votre Identifiant Machine (HWID)

Cliquez sur n'importe quel onglet dans le menu à gauche (par exemple : Facturation ou Stock).

Une fenêtre de sécurité va s'ouvrir. C'est la page d'activation. Vous y verrez un code unique appelé Identifiant Machine.

3

Faites votre demande via Facebook

Envoyez-nous un simple message privé sur notre page Facebook avec les informations suivantes :

Veuillez copier/coller et remplir ce formulaire dans votre message :
  • Nom & Prénom
  • Adresse Email
  • Nom de votre Entreprise
  • Numéro de Téléphone
  • Votre Identifiant Machine (HWID) généré à l'étape 2.
Envoyer mon message sur Messenger
4

Activation de l'application

Notre développeur générera votre clé de licence personnalisée et vous l'enverra en réponse sur Messenger.

Copiez cette clé, collez-la dans la case Clé de Licence de l'application (vue à l'étape 2), et cliquez sur "Enregistrer". Votre application est déverrouillée !

5

Connexion à l'application

Une fois la licence validée, l'application vous demandera de vous connecter pour sécuriser l'accès à vos données.

Utilisez ces identifiants administrateur par défaut :
  • Nom d'utilisateur : admin
  • Mot de passe : admin
6

Configuration Initiale (Paramètres)

Félicitations, vous êtes connecté au tableau de bord ! Pour commencer, personnalisons vos documents.

Naviguez jusqu'à l'onglet Paramètres (dans le menu de gauche) pour :

  • Importer le logo de votre entreprise.
  • Entrer votre adresse, email et téléphone.
  • Renseigner vos informations légales (RIDET, RCS).
7

Sécurisez votre compte (Mon Profil)

Il est impératif de modifier les identifiants par défaut pour garantir la sécurité de votre logiciel comptable.

Naviguez vers l'onglet Mon Profil (en bas à gauche) et suivez cette méthode standard :

  1. Modifiez votre Nom d'utilisateur.
  2. Entrez le mot de passe actuel (admin).
  3. Entrez votre Nouveau mot de passe (celui de votre choix).
  4. Confirmez le nouveau mot de passe et cliquez sur Enregistrer.
